# Yutrel

个人出于兴趣开发的vulkan玩具工程，利用vulkan-hpp封装进行开发，使用Xmake作为构建系统，仅在windows平台上进行开发与测试，shader语言选用hlsl

History分支为2023年12月-2024年4月的开发历史，现已基本废弃，仅用作参考及留档

# 关键词

Vulkan, Xmake，HLSL，vulkan-hpp
